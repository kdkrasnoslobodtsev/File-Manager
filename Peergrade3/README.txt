Приветствую Вас в программе "Файловый менеджер"
Здесь я представлю описание данного проекта
Пройдемся по функционалу:

1) Просмотр списка дисков компьютера и выбор диска:
Пользователю будет выведен список дисков и ему следует выбрать тот, с каким он будет работать (впоследствии диск можно поменять)

2) Переход в другую директорию (выбор папки):
Эта функция станет доступна пользователю только после выбора диска
Следует ввести путь до директории относительно диска
Пример (выбран диск C:\):
Нужная директория C:\users\MyName\MyFiles
Следует ввести в консоль users\MyName\MyFiles
Если требуется работа с файла находящимся непосредственно на диске, а не в его директориях, следует ввести пустую строку

3) Просмотр списка файлов в директории:
Эта функция станет доступна пользователю только после выбора диска и директории
Пользователю покажут список файлов той директории, которую он выбрал

4) Вывод содержимого текстового файла в консоль в кодировке UTF-8:
Эта функция станет доступна пользователю только после выбора диска и директории
В консоль будет выведено содержимое файла в кодировку UTF-8, который выберет пользователь. Файл должен находиться в директории, выбранной пользователем изначально

5) Вывод содержимого текстового файла в консоль в выбранной пользователем кодировке(предоставляется не менее трех вариантов):
Эта функция станет доступна пользователю только после выбора диска и директории
В консоль будет выведено содержимое файла в трех возможных кодировках(Unicode, UTF-32, ASCII), который выберет пользователь. Файл должен находиться в директории, выбранной пользователем изначально

6) Копирование файла:
Эта функция станет доступна пользователю только после выбора диска и директории
Происходит копирование файла в ту же директорию, что и изначальный файл. Пользователь может сам задать имя новому файлу.

7) Перемещение файла в выбранную пользователем директорию:
Эта функция станет доступна пользователю только после выбора диска и директории
Происходит перемещение файла, который выбрал пользователь, в другую директорию. Файл должен находиться в директорией, выбранной пользователем изначально. Новую директорию пользователь выбирает сам.

8) Удаление файла:
Эта функция станет доступна пользователю только после выбора диска и директории
Происходит удаление файла, который выбрал пользователь. Файл должен находиться в директории, выбранной пользователем изначально

9) Создание простого текстового файла в кодировке UTF-8:
Эта функция станет доступна пользователю только после выбора диска и директории
Пользователь создает файл в кодировке UTF-8 в директории, указанной пользователем изначально. Дается возможность что-нибудь в него записать.

10) Создание простого текстового файла в выбранной пользователем кодировке(предоставляется не менее трех вариантов):
Эта функция станет доступна пользователю только после выбора диска и директории
Пользователь создает файл в одной из предложенных кодировок(Unicode, UTF-32, ASCII) в директории, указанной пользователем изначально. Дается возможность что-нибудь в него записать.

11) Конкатенация содержимого двух или более текстовых файлов и вывод результата в консоль в кодировке UTF-8:
Эта функция станет доступна пользователю только после выбора диска и директории
Пользователю предлагается читать информацию из файлов в кодировке UTF-8 из директории, выбранной пользователем изначально, пока он не захочет завершить

12) Вывод всех файлов в текущей директории по заданной маске*:
Эта функция станет доступна пользователю только после выбора диска и директории
Пользователь вводит маску для поиска файлов, программа выводит названия файлов из выбранной пользователем директории, которые подходят под маску

13) Вывод всех файлов в текущей директории и всех её поддиректориях по заданной маске:
Эта функция станет доступна пользователю только после выбора диска и директории
Пользователь вводит маску для поиска файлов, программа выводит названия файлов из выбранной пользователем директории и ее поддиректорий, которые подходят под маску

14) Копирование все файлы из директории и всех её поддиректорий по маске в другую директорию:
Эта функция станет доступна пользователю только после выбора диска и директории
Пользователь вводит маску для поиска файлов, программа копирует, подходящие под маску файлы, из выбранной пользователем изначально директории и ее поддиректорий в другую директорию, которую выбирает пользователь(если ее не существует, она создается). Если в этой директории уже есть фалы с такими же названиями, как и у копируемых, то пользователю предоставляется выбор, оставить старые файлы без изменений или заменить их на новые

Справка:
Маска файла – это представление имени и расширения файла общими символами. Пример маски: *a?.txt
